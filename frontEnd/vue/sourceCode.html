# 挂载的实现

mountComponent 核心就是先实例化一个渲染Watcher，在它的回调函数中会调用 updateComponent 方法，
在此方法中调用 vm._render 方法先生成虚拟 Node，最终调用 vm._update 更新 DOM

# createComponent

渲染一个原生组件，它在渲染一个组件的时候的 3 个关键逻辑：构造子类构造函数，安装组件钩子函数和实例化 vnode。
